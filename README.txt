DESCRIPTION
This is a website in which users can check reviews and suggestions about some products and also order them.
......................................................................................................................
FEATURES
-User can register in the website.
-User can login to the website.
-User can check on all the products displayed in the home page, as well as on categories they are seperated.
-User can check details about products, including product code, amount in ml, price and review about that product.
-User can order the products
-User can send requests when ordering the product.
-User can check his/her profile to see their personal information and orders.
-User can update his/her personal informations.
-User can send requests/feedback by contacting the page.
-User can search on their order and view details about them.
